# Leetcode Analytics

A full-stack web application that analyzes LeetCode problem-solving data and provides interactive visualizations.

## Features

- User Authentication
- Dashboard
- Problem Statistics
- Difficulty Analysis
- Language-wise Analysis
- Interactive Charts
- Jupyter Notebook Analysis

## Tech Stack

### Frontend
- React.js
- CSS
- JavaScript

### Backend
- FastAPI
- Python
- Pandas

### Data Analysis
- Jupyter Notebook
- Matplotlib

## Project Structure

```
Leetcode-Analytics/
│
├── frontend/

```

## Installation

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Leetcode-Analytics.git
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
